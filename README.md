# Trinca Brasil — Site

Site do **Projeto Trinca Brasil** (concerto e oficina sobre a história do choro):
apresentação institucional, captação de patrocínio via Lei Federal de Incentivo à
Cultura (Lei Rouanet) e área de solicitação de certificado da oficina.

## Conteúdo

- `index.html` — o site completo, em arquivo único (HTML/CSS/JS, imagens embutidas).

## Como ver localmente

É só abrir o `index.html` no navegador (duplo clique). Não precisa de servidor.

## Publicar

Por ser um site estático de arquivo único, pode ser publicado em qualquer lugar:

- **GitHub Pages:** Settings → Pages → Branch `main` / pasta raiz.
- **Vercel / Netlify:** conectar este repositório; sem configuração de build.

## Próximos passos (em desenvolvimento)

- Gravar as solicitações de certificado no **Supabase** (hoje ficam no navegador).
- Automação: Supabase → n8n → serviço gerador → envio do certificado por e-mail.
